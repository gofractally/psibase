# Colors

These colors come from our Figma and are configured for use with Tailwind in our project. Tailwind is restricted to these colors. If we need others, they can be added in the appropriate `tailwind.config.js` files. But we should only add colors if they are on brand. If your Tailwind IntelliSense is properly configured, it will only suggest these colors to you.

To use these colors, use the relevant tailwind class with the `tw:color` listed below. For example, if you want to make text the Success 300 shade, you would use the class `text-green-300`.
